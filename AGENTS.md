## 开发规范

### 代码规范
- 前端开发不要使用 emoji
- 使用 MIT 或 Apache 开源协议的符号库
- 不要使用 GPL 协议的库

### 文件管理
- 进行比较大的更改时，记得更新本 AGENTS.md 文件

## 技术栈
- VitePress 1.3.4
- Vue 3
- TypeScript

## 项目结构
```
docs/
  .vitepress/       # VitePress 配置
    config.ts      # 主配置文件
    theme/         # 主题配置
  guide/           # 用户指南文档
  public/          # 静态资源
```
